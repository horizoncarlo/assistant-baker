Digital assistant app for the [221B Baker Street boardgame](https://boardgamegeek.com/boardgame/1275/221b-baker-street-the-master-detective-game) from the 70s. Easier case referencing, clue tracking, and solution guessing.

See https://horizoncarlo.github.io/assistant-baker/ for the deployed version.

Or just download the HTML `index.html` page and double click it.
